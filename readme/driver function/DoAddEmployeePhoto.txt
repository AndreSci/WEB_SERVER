
Вызов - http://127.0.0.1:8066/DoAddEmployeePhoto
(POST)

json данные
{
    "id": id
    "img64": (base64 jpg, png, max-size 1080px, файл должен быть менее 2мб.)

    Возможные поля
    "deviceId":      (если нужно применить для одного устройства указываем его Id из базы)
    "isGuest":       (является ли гостем)
    "idCardNum":     (номер карты)
    "qrCode":        (значение qr кода)

}

отвечает json
{"RESULT": "SUCCESS\ERROR", "DESC": "", "DATA": ""}

SUCCESS: Успешное выполнение функции
ERROR: На данного владельца карты уже есть запись
ERROR: Была добавлена запись в базу, но при добавлении фото произошла ошибка
ERROR: ErrorOnServer
ERROR: Не определенный результат
ERROR: error_with_base_helper

SUCCESS_GUEST: Если запрос без фото (определяет как попытка создания сотрудника с дальнейшей процедурой добавления фото через терминал )
